How the tech works

The headset sends a moving picture to each eye, creating a 3d depth effect. The textures of objects are clearly visible, including colors.

We offer two tiers of the setup: a tethered version with an Oculus headset connected to a PC. This would be setup in a room or large space with upto 4 sensors placed around evenly. The advantage of this setup is a clearer picture and more realistic motion. Boundaries are set using the software so that the client never runs into a physical object in the space.



Our mobile tier is a Samsung Android phone. This is a tetherless headset which relies solely on an Samsung Android phone in a VR headset. This tier is more budget friendly. Although the picture is not as clear, and the device will have to be charged when used frequently. No sensors are supplied with this setup, however a game controller is included so that the client may explore the model freely.

Our tethered tier setup includes a custom built PC with a Quad Core AMD Ryzen 3rd Generation 3400, an Nvidia GTX 1060 Graphics Card, and 16 GB of DDR4 Memory. This hardware will guarantee a flawless, smooth VR experience. We include a complementary maintenance service upon purchasing this tier.
